<h2>🛠️ Installation Steps:</h2>

<p>1. clone cette projet</p>

```
git clone https://github.com/Herimanjaka007/weather-app.git weather-app
```

<p>2. installer les dependances</p>

```
npm install
```

<p>3. lancer le serveur</p>

```
npm run dev
```
